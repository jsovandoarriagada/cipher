## Getting started

👉 Clone the repository:
   ```sh
   git clone https://github.com/jsovandoarriagada/purple-man-gaming-v2.git
   ```
👉 Install npm packages:
   ```sh
   npm install
   ```
👉 Run the app:
   ```sh
   npm start
   ```
